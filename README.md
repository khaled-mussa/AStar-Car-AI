# A* Car Pathfinding Project

## 📌 Description
This project implements an AI-controlled car that navigates from a start position to a goal using the A* pathfinding algorithm. The environment is represented as a grid where obstacles are detected, and the optimal path is calculated.

## 🎯 Objectives
- Implement A* algorithm
- Represent environment using grid system
- Allow AI to avoid obstacles
- Simulate autonomous navigation

## ⚙️ Technologies Used
- Unity (C#)
- A* Algorithm

## 🧠 How it Works
- The environment is divided into nodes (grid)
- Each node is marked as walkable or not
- The A* algorithm searches for the shortest path
- The car follows the calculated path to reach the goal
